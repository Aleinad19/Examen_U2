# Mi Portafolio

Portafolio personal desarrollado con:

- HTML
- CSS
- JavaScript
- Docker

## Características

- Diseño responsive
- Animaciones suaves
- Filtro de proyectos
- Validación de formulario
- Dockerizado con Nginx

## Ejecutar con Docker

```bash
docker build -t mi-portafolio .
docker run -d -p 8080:80 mi-portafolio
